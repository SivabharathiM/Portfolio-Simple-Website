# Personal Portfolio Website

This is a simple personal portfolio website created using **HTML** and **CSS**. It is designed to present basic information about me and provide a way for visitors to contact me.

## Pages

* **Home Page** – Introduces me and gives a brief overview of my portfolio.
* **Contact Page** – Allows visitors to reach out to me through contact information or a form.

## Technologies Used

* HTML5
* CSS3

## Project Purpose

The purpose of this project is to practice basic **web development skills** by building a simple multi-page website using only HTML and CSS.

## How to Run

1. Download or clone the repository.
2. Open the `index.html` file in your browser.
